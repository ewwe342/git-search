# Git Search 2026 ❄️

**Git Search** — это эстетичный терминальный клиент (TUI) для продвинутого поиска по репозиториям GitHub. Программа сочетает в себе мощь API GitHub и праздничную атмосферу с анимацией падающего снега.

### Основные возможности
* **Умный поиск:** Находит репозитории быстрее и точнее стандартного поиска за счет поддержки квалификаторов (например, `stars:>1000`).
* **Новогодний интерфейс:** Интерактивное меню с эффектом снегопада в консоли.
* **Описание проектов:** Отображение количества звезд и краткого описания прямо в таблице.

### Технологии
* **Python 3.10+**
* **Textual:** Современный TUI фреймворк.
* **Requests:** Работа с API.
